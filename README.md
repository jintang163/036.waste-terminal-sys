## 技术栈

### 前端移动端
- **框架: Flutter 3.x
- **离线数据库: sqflite
- **网络请求: dio
- **状态管理: Provider + GetX
- **蓝牙/USB: 打印机、地磅连接
- **扫码: qr_code_scanner
- **图片: image_picker + flutter_image_compress

### 后端
- **框架: Spring Boot 2.7.x
- **ORM: MyBatis Plus 3.5.x
- **数据库: MySQL 8.0
- **缓存: Redis 6.x
- **消息队列: RocketMQ 4.9.x
- **文件存储: MinIO
- **网关: Netty (地磅串口网关)
- **安全: 国密SM2/SM4
- **二维码: zxing

## 核心功能模块

### 1. 危废入库（离线称重、扫码、拍照）
- 离线状态下完成危废入库
- 蓝牙连接地磅获取重量 / 手动输入
- 扫描容器码或自动生成新码
- 拍照留档（本地压缩存储）
- 数据存入 SQLite 离线库
- 网络恢复后自动同步
- 离线打印标签（蓝牙打印机）

### 2. 出库转移与电子联单（离线发起）
- 扫描容器标签，从本地缓存调取入库信息
- 选择接收单位、运输车辆
- 自动生成电子联单数据
- 离线状态存本地，有网后自动上报
- 联单二维码展示、打印
- 联单状态管理（待签收→已签收）

### 3. 库存实时监控与离线预警
- 首页展示危废库存量、贮存期限、库容占用率
- 后端定时扫描库存，超期/超量预警
- 极光推送/短信通知
- 离线盘点：扫描容器标签，生成差异报告
